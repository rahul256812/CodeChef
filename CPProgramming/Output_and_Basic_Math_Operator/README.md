# Output and Basic Math Operators (C++ / Competitive Programming)

This folder contains introductory competitive programming exercises, multiple-choice questions (MCQs), and basic C++ challenges focusing on standard output (`cout`), newlines (`endl`, `\n`), and basic arithmetic operators. All programs utilize `using namespace std;` for simplicity.

## Contents

### 1. Multiple-Choice Questions (MCQs)
* **File:** [`mcq.txt`](./mcq.txt)
* **Description:** A set of 16 multiple-choice questions testing core concepts of standard output stream, operator precedence, string and character concatenation, and the usage of newlines in C++.

### 2. Print 108
* **File:** [`print_108.cpp`](./print_108.cpp)
* **Description:** A C++ program that prints the value `108` by multiplying the numbers `9` and `12`.
* **Output:**
  ```text
  108
  ```

### 3. Learn Coding on CodeChef
* **File:** [`learn_coding.cpp`](./learn_coding.cpp)
* **Description:** Prints "Learn Coding on CodeChef" to the console.
* **Output:**
  ```text
  Learn Coding on CodeChef
  ```

### 4. Print I love CodeChef
* **File:** [`love_codechef.cpp`](./love_codechef.cpp)
* **Description:** A C++ program that outputs a multi-line message exactly as specified.
* **Output:**
  ```text
  I 
  love 
  CodeChef
  ```

### 5. Print Squares
* **File:** [`print_squares.cpp`](./print_squares.cpp)
* **Description:** Computes and prints the square of numbers from 1 to 5 on separate lines using multiplication.
* **Output:**
  ```text
  1 - 1
  2 - 4
  3 - 9
  4 - 16
  5 - 25
  ```

### 6. Arithmetic Operations
* **File:** [`arithmetic_operations.cpp`](./arithmetic_operations.cpp)
* **Description:** Performs and prints the results of addition, subtraction, division, and multiplication between the numbers `8` and `4`.
* **Output:**
  ```text
  12
  4
  2
  32
  ```

---

## How to Compile and Run

To compile and run any of the C++ programs, open your terminal, navigate to this folder, and run:

```bash
g++ -O3 <filename.cpp> -o <output_name>
./<output_name>
```

**Example:**
```bash
g++ -O3 print_squares.cpp -o print_squares
./print_squares
```
