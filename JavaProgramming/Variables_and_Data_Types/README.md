# Variables and Data Types (Java)

This folder contains introductory Java programming exercises, multiple-choice questions (MCQs), and basic Java programs focusing on variable declaration, initialization, arithmetic operations, and precision data types (integers, doubles, characters, booleans, longs).

## Contents

### 1. Multiple-Choice Questions (MCQs)
* **File:** [`mcq.txt`](./mcq.txt)
* **Description:** A set of 8 multiple-choice questions testing fundamental Java syntax, data types (`char`, `boolean`, `double`, `long`), variable declaration, and basic arithmetic precision.

### 2. Divide Two Variables
* **File:** [`DivideVariables.java`](./DivideVariables.java)
* **Description:** Declares two integer variables `x = 20` and `y = 6`, performs integer division (`x / y`), and prints the result.
* **Output:**
  ```text
  3
  ```

### 3. Convert Temperature
* **File:** [`ConvertTemperature.java`](./ConvertTemperature.java)
* **Description:** Declares a variable `temperature = 25.5` (Celsius), converts it to Kelvin (Celsius + 273), and prints both in a specified format.
* **Output:**
  ```text
  Celsius - 25.5
  Kelvin - 298.5
  ```

---

## How to Compile and Run

To compile and run any of the Java programs, open your terminal, navigate to this folder, and run:

```bash
javac <ClassName.java>
java <ClassName>
```

**Example:**
```bash
javac ConvertTemperature.java
java ConvertTemperature
```
