# Print Statement and Java Syntax

This folder contains introductory Java programming exercises, multiple-choice questions (MCQs), and basic Java syntax challenges focusing on standard output (`System.out.println`), escape sequences, and basic arithmetic operators.

## Contents

### 1. Multiple-Choice Questions (MCQs)
* **File:** [`mcq.txt`](./mcq.txt)
* **Description:** A set of 16 multiple-choice questions testing fundamental Java syntax, standard output methods, escape characters (`\n`), string concatenation, and code structure.

### 2. Print 108
* **File:** [`Print108.java`](./Print108.java)
* **Description:** A Java program that computes and prints the number `108` using arithmetic multiplication on `9` and `12`.
* **Output:**
  ```text
  108
  ```

### 3. Learn Coding on CodeChef
* **File:** [`LearnCoding.java`](./LearnCoding.java)
* **Description:** Prints "Learn Coding on CodeChef" to the console.
* **Output:**
  ```text
  Learn Coding on CodeChef
  ```

### 4. Print I love CodeChef
* **File:** [`LoveCodeChef.java`](./LoveCodeChef.java)
* **Description:** Prints each word of the sentence "I love CodeChef" on a new line.
* **Output:**
  ```text
  I
  love
  CodeChef
  ```

### 5. Print Squares
* **File:** [`PrintSquares.java`](./PrintSquares.java)
* **Description:** Outputs the squares of numbers from 1 to 5 on separate lines using multiplication.
* **Output:**
  ```text
  1 - 1
  2 - 4
  3 - 9
  4 - 16
  5 - 25
  ```

---

## How to Compile and Run

To compile and run any of the Java programs, open your terminal, navigate to this folder, and run:

```bash
javac <ClassName.java>
java <ClassName>
```

**Example:**
```bash
javac PrintSquares.java
java PrintSquares
```
