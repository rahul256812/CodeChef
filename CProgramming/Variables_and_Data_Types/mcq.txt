Q1. Variable Initialization
What does this C code print?
#include <stdio.h>
int main() {
    int number = 10;
    printf("%d", number);
    return 0;
}
A) 10
B) number
C) 00
D) Error
Answer: A

Q2. Data Type Precision
What is the output of the given C code?
#include <stdio.h>
int main() {
    float value = 7.8;
    printf("%f", value);
    return 0;
}
A) 7
B) 7.800000
C) 8
D) 7.5
Answer: B

Q5. Variable Scope
What does the given C code print?
#include <stdio.h>
int main() {
    int num = 5;
    {
        int num = 10;
        printf("%d ", num);
    }
    printf("%d", num);
    return 0;
}
A) 10 5
B) 5 10
C) 10 10
D) Error
Answer: A
