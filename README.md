# CodeChef Programming Solutions & Exercises

Welcome to my personal CodeChef programming workspace! This repository contains a structured collection of solved coding problems, fundamental programming challenges, and conceptual multiple-choice questions (MCQs) spanning four key paths: **C**, **C++ (Competitive Programming)**, **Java**, and **Python**.

---

## 📂 Repository Structure

The workspace is organized into language-specific folders, each subdivided into conceptual modules featuring practical coding exercises and conceptual quizzes (`mcq.txt`).

```text
CodeChef/

```

## 🛠️ Folder Breakdown

### 1. 📘 [C Programming](./CProgramming/)
Designed to build a robust foundation in memory operations, low-level variables, and print formatting using standard C syntax.
* **[Introduction, Output & Math Operators](./CProgramming/Introduction_Output_and_Math_Operators/)**: Multi-line outputs, custom printf formats, basic arithmetic operations, and an extensive 22-question MCQ bank.
* **[Variables and Data Types](./CProgramming/Variables_and_Data_Types/)**: Swapping algorithms, float precision conversions (Celsius/Fahrenheit), BMI calculators, and an 11-question MCQ bank.

### 2. 🚀 [Competitive Programming (C++)](./CPProgramming/)
Uses C++ for high-performance and standard competitive programming templates.
* **[Output and Basic Math Operators](./CPProgramming/Output_and_Basic_Math_Operator/)**: Practice with `cout`, `endl`, nested operations, multiplication challenges, and a 16-question competitive programming MCQ bank.

### 3. ☕ [Java Programming](./JavaProgramming/)
Focused on object-oriented structures, strict type safety, class declarations, and standard Java API syntax.
* **[Print Statement & Java Syntax](./JavaProgramming/Print_Statement_and_Java_Syntax/)**: Class structures (`public class Main`), `System.out.println`, string concatenation, and a 16-question MCQ bank.
* **[Variables and Data Types](./JavaProgramming/Variables_and_Data_Types/)**: Handling of `double`/`long` types, division truncation, temperature transformations, and an 8-question MCQ bank.

### 4. 🐍 [Python Programming](./PythonProgramming/)
Highlights clean, readable, dynamic typing and powerful high-level string manipulations.
* **[Introduction, Output & Math Operators](./PythonProgramming/Introduction_Output_and_Math_Operators/)**: Star pattern outputs, f-strings (`print(f"...")`), arithmetic evaluations, and a 16-question Python-specific MCQ bank.

## 🚀 Getting Started

To run any of the programs, navigate into the respective directory and follow the compile/run instructions outlined in each subfolder's detailed `README.md` file.

### Quick Reference

* **C Programs:**
  ```bash
  gcc filename.c -o output && ./output
  ```
* **C++ Programs:**
  ```bash
  g++ -O3 filename.cpp -o output && ./output
  ```
* **Java Programs:**
  ```bash
  javac ClassName.java && java ClassName
  ```
* **Python Programs:**
  ```bash
  python3 filename.py
  ```
