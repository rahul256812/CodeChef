# Introduction, Output, and Math Operators (Python)

This folder contains introductory Python programming exercises, multiple-choice questions (MCQs), and basic syntax challenges focusing on the standard `print()` function, newlines, f-strings, and basic arithmetic operators.

## Contents

### 1. Multiple-Choice Questions (MCQs)
* **File:** [`mcq.txt`](./mcq.txt)
* **Description:** A set of 16 multiple-choice questions testing fundamental Python syntax, standard output options (`end=`), escape characters (`\n`), string concatenation, and code structure.

### 2. Print 108
* **File:** [`print_108.py`](./print_108.py)
* **Description:** A Python program that computes and prints the number `108` using arithmetic multiplication on `9` and `12`.
* **Output:**
  ```text
  108
  ```

### 3. Learn Coding on CodeChef
* **File:** [`learn_coding.py`](./learn_coding.py)
* **Description:** Prints "Learn Coding on CodeChef" to the console.
* **Output:**
  ```text
  Learn Coding on CodeChef
  ```

### 4. Print Right Angled Triangle
* **File:** [`right_triangle.py`](./right_triangle.py)
* **Description:** Prints a right-angled triangle pattern of stars of height 5.
* **Output:**
  ```text
  *
  **
  ***
  ****
  *****
  ```

### 5. Print I love CodeChef
* **File:** [`love_codechef.py`](./love_codechef.py)
* **Description:** Prints each word of the sentence "I love CodeChef" on a new line.
* **Output:**
  ```text
  I
  love
  CodeChef
  ```

### 6. Print Squares
* **File:** [`print_squares.py`](./print_squares.py)
* **Description:** Outputs the squares of numbers from 1 to 5 on separate lines using f-strings and multiplication.
* **Output:**
  ```text
  1 - 1
  2 - 4
  3 - 9
  4 - 16
  5 - 25
  ```

### 7. Print Pattern
* **File:** [`print_pattern.py`](./print_pattern.py)
* **Description:** Prints a 4x4 square of stars.
* **Output:**
  ```text
  ****
  ****
  ****
  ****
  ```

---

## How to Run

To run any of the Python programs, open your terminal, navigate to this folder, and execute using your Python 3 interpreter:

```bash
python3 <filename.py>
```

**Example:**
```bash
python3 print_squares.py
```
